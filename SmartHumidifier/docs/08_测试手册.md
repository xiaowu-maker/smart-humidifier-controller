## 1. 测试策略

### 1.1 测试金字塔

本项目采用经典测试金字塔策略，从底层逐级向上验证，确保每一层质量后进入下一层：

```
              ╱╲
             ╱  ╲             验收测试 (2 项)
            ╱    ╲
           ╱      ╲          系统测试 (6 项)
          ╱        ╲
         ╱          ╲       集成测试 (8 项)
        ╱            ╲
       ╱══════════════╲    单元测试 (30+ 项)
      ╱                ╲
     ╱                  ╲
```

| 层级 | 数量目标 | 执行频率 | 责任方 | 通过标准 |
|------|---------|---------|--------|---------|
| **单元测试** | 30+ | 每次提交 | 开发者 | 100% 通过，覆盖率 > 80% |
| **集成测试** | 8 | 每日构建 | 开发者 | 100% 通过 |
| **系统测试** | 6 | 版本发布前 | 测试团队 | 100% 通过 |
| **验收测试** | 2 | 版本发布 | PM + QA | 100% 通过 |

### 1.2 测试环境说明

#### 硬件测试环境（HIL — Hardware-in-the-Loop）

| 环境项 | 配置 | 用途 |
|--------|------|------|
| **主控** | ESP32 DevKit V1 (CP2102) × 2 | 1 块运行被测固件，1 块运行硬件模拟器 |
| **传感器** | AHT20 × 2（备用 1 个用于破坏性测试） | 温湿度数据源 |
| **显示** | 0.96寸 OLED SSD1306 × 1 | 显示验证 |
| **执行器** | 单路继电器 5V × 1 | 加湿器控制 |
| **指示器** | 共阴 RGB LED × 1 | 状态指示 |
| **报警** | 有源蜂鸣器 5V × 1 | 报警输出 |
| **输入** | 轻触按键 × 2 | 用户交互 |
| **电源** | 5V/2A USB 适配器 | 系统供电 |
| **辅助工具** | 数字万用表、逻辑分析仪、USB 转 TTL 模块 | 调试与监控 |

#### 软件测试环境

| 工具 | 版本 | 用途 |
|------|------|------|
| PlatformIO | ≥ 6.0 | 嵌入式编译与烧录 |
| Arduino 框架 | ≥ 3.0.0 | ESP32 底层驱动 |
| Unity Test | ≥ 2.5.2 | 单元测试框架（嵌入式） |
| Google Test | ≥ 1.12 | PC 端单元测试（模拟层） |
| Python 3 + pytest | ≥ 3.10 / ≥ 7.0 | 集成与系统测试脚本 |
| Minicom / PuTTY | — | 串口监视与日志采集 |
| Saleae Logic | 2.4+ | I2C 总线分析 |

#### 环境准备步骤

```bash
# 1. 安装 PlatformIO CLI
pip install platformio

# 2. 安装 Python 测试依赖
pip install pyserial pytest pytest-embedded

# 3. 克隆固件仓库
cd SmartHumidifier/firmware

# 4. 安装依赖库（通过 platformio.ini）
pio lib install

# 5. 验证测试环境
pio test --without-uploading
```

### B. Python 自动日志采集脚本

```python
#!/usr/bin/env python3
"""
SmartHumidifier Test Logger
自动采集串口日志并保存到文件，用于 72 小时稳定性测试。
"""
import serial
import datetime
import sys
import os
import signal

LOG_FILE = "shc_test_log_{}.txt".format(
    datetime.datetime.now().strftime("%Y%m%d_%H%M%S")
)
SERIAL_PORT = "COM3"  # 根据实际情况修改
BAUD_RATE = 115200

running = True

def signal_handler(sig, frame):
    global running
    print("\n[Logger] Stopping...")
    running = False

def main():
    signal.signal(signal.SIGINT, signal_handler)
    
    ser = serial.Serial(SERIAL_PORT, BAUD_RATE, timeout=1)
    log_fd = open(LOG_FILE, "w", encoding="utf-8")
    
    log_fd.write(f"=== SmartHumidifier Test Log ===\n")
    log_fd.write(f"Start Time: {datetime.datetime.now()}\n")
    log_fd.write(f"Port: {SERIAL_PORT} @ {BAUD_RATE} bps\n")
    log_fd.write("=" * 50 + "\n")
    log_fd.flush()
    
    print(f"[Logger] Recording to {LOG_FILE} ...")
    
    line_count = 0
    while running:
        try:
            line = ser.readline().decode("utf-8", errors="replace").strip()
            if line:
                timestamp = datetime.datetime.now().strftime("%H:%M:%S.%f")[:-3]
                log_line = f"[{timestamp}] {line}"
                print(log_line)
                log_fd.write(log_line + "\n")
                line_count += 1
                
                # 每 100 行刷一次缓冲区
                if line_count % 100 == 0:
                    log_fd.flush()
                    
        except serial.SerialException:
            print("[Logger] Serial disconnected, reconnecting...")
            try:
                ser.close()
                ser = serial.Serial(SERIAL_PORT, BAUD_RATE, timeout=1)
            except:
                pass
        except Exception as e:
            print(f"[Logger] Error: {e}")
    
    log_fd.write(f"\n=== Log Ended ===\n")
    log_fd.write(f"End Time: {datetime.datetime.now()}\n")
    log_fd.write(f"Total Lines: {line_count}\n")
    log_fd.close()
    ser.close()
    print(f"[Logger] Saved {line_count} lines to {LOG_FILE}")

if __name__ == "__main__":
    main()
```

// 测试辅助：模拟湿度值递增/递减
// 在单元测试中替代 SensorManager::readSensor()
void simulate_humidity_ramp(float start, float end, float step, int delay_ms) {
    for (float h = start; (step > 0) ? (h <= end) : (h >= end); h += step) {
        controller.setSimulatedHumidity(h);
        controller.loop();
        Serial.printf("[TEST] Simulated humidity: %.1f%%, Relay: %s\n",
                      h, controller.isRelayOn() ? "ON" : "OFF");
        delay(delay_ms);
    }
}
